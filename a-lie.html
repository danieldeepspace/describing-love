爱是一个谎言

{% image "escalator-ride.jpg", "escalator", "media" %}

很多时候，爱被扭曲、被滥用、被背叛。而这些行径，却往往在爱的名义下进行。

请容我提出一点小小的真相：虽然以爱为名的谎言层出不穷，但爱本身却是谎言的对立面。

爱至真至诚，永不止息。
爱浓烈纯洁。
爱无处不在。

也许最后一条描述会让你疑惑。也许你已经竭尽所能，四处寻访，竭尽全力地去寻找爱，但最终换来的却只是伤心失望。

{% image "man-with-glasses.jpg", "man with glasses", "media" %}

还谈什么爱无处不在呢？

其实，那个爱你的，就在你身边。他已经在爱中把自己献给了你。上帝就是爱，他的爱是可以触及的，认识他也是可能的。

听听当上帝的爱在运行时，生命是什么样子的。

亲爱的弟兄姊妹，我们应当彼此相爱，因为爱是从上帝来的。凡有爱心的人都是从上帝生的，并且认

识上帝。 没有爱心的人不认识上帝，因为上帝就是爱。 上帝差遣祂独一的儿子到世上来，是为了叫我们靠着祂得到生命。这就显明了上帝对我们的爱。 ——约翰一书4:7-9
{% image "watching-the-sunrise.jpg", "watching the sunrise", "image-wrapper" %}

这就是那个令人震惊的现实，上帝一直爱着我们，他不仅要给我们他的爱，也要献出他自己。

他爱你，他愿意和你在一起，这会让你的生命焕然一新——那感觉就像是从头来过，再生一次。

无论是谁，他若承认耶稣是上帝的儿子，上帝必住在他里面，他也住在上帝里面。 我们已经认识并且相信了上帝对我们的爱。 ——约翰一书4：15-16
